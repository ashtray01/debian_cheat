# 🐧 Debian Cheat Шпаргалка — Dev & Deploy

Единый HTML-файл (self-contained) с самыми важными командами Linux для разработки и деплоя.

> ⚠️ В Debian **sudo** может быть не установлен по умолчанию.
> Используй `su -` для перехода на root или установи sudo:
> `apt update && apt install sudo -y && usermod -aG sudo $USER`

## Возможности

- **25 разделов**, 257 команд — от `apt` до `kubectl`
- **Tooltip-подсказки** — наведи на команду, чтобы увидеть пояснение
- **Адаптивная сетка** — от 1 до 4 колонок в зависимости от ширины экрана
- **Поиск** (Ctrl+F не нужен) — встроенное поле поиска со скрытием пустых разделов
- **Бейджи** — HOT, PROD, DEV, SEC, DB, NET для быстрой навигации
- **Никаких зависимостей** — открывай в браузере и пользуйся
